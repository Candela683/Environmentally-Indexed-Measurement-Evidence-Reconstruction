# Local Secret Files

Place API keys here only on the local machine. These files are local runtime credentials and are not part of the distributed source package.

Example files:

- `openai_api_key.txt`
- `ollama_api_key.txt`
- `google_maps_api_key.txt`

The code also supports environment variables such as `OPENAI_API_KEY` and `GOOGLE_MAPS_API_KEY`.
